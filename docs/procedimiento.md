\## 🧪 ¿Cómo usar todo esto paso a paso?



1\. Creá una carpeta, por ejemplo `C:\\TP\_Auditoria`.

2\. Guardá los dos `.bat` ahí.

3\. Ejecutá `setup\_project.bat` → se crean las carpetas.

4\. En GitHub: creá un nuevo repositorio (vacío).

5\. En tu PC, dentro de la carpeta:

&#x20;  ```bash

&#x20;  git init

&#x20;  git remote add origin https://github.com/tuusuario/NOMBRE\_REPO.git

&#x20;  git add .

&#x20;  git commit -m "chore: estructura inicial"

&#x20;  git push -u origin main
